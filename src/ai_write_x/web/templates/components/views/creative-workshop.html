创作主题

支持自定义标题或直接使用系统热搜话题

勾选“自定义话题”后可输入标题，留空则自动采用热搜话题

模板与结构

选择最适合文章风格的模板，快速匹配内容结构

自定义话题下可指定模板分类，未选择时由系统自动匹配

可以从模板管理中维护模板内容

参考素材

提供参考链接和借鉴比例，帮助模型更好地掌握素材

链接需以 http(s):// 开头，最多支持 10 个

控制文章对参考素材的借鉴程度

准备好后点击“开始生成”即可自动完成整篇文章的写作，可随时使用“停止生成”中止任务。

使用说明

  • 确保在“系统设置 → 大模型 API”中配置可用的 OpenAI 兼容接口和 Key
  • 支持填写自建或代理的 API Base 地址
  • 如需结合外部资料，可粘贴参考链接并设置借鉴比例
  • 模板配置来自模板管理，可根据行业场景自定义
